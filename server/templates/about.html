Concuss.js

What am I like?

Concuss.js is a JavaScript application that gathers information that tests a subject's cognition, reflex, and balance. This application is planned to be distributed via a mobile platform to detect if users have a concussion. The data gathered is sent to a RethinkDB database so it can be queried and analysed at a later time. The tests work by aquiring user information, user test data, and the classification of the test (such as baseline, pre-season, post-injury) so that deductions can be made on whether a particular user has a concussion.
