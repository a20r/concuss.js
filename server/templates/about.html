Concuss.js

What am I like?

Concuss.js is a JavaScript application that will be used to gather information that tests a subject's cognition, reflex, and balance. This application is planned to be distributed via a mobile platform and hopes to be used to classify if a subject has a concussion. The simple games that the subjects will play can be used as a base line to test whether, after some collision, if the person has suffered a head injury.

This application is being developed at the University of Notre Dame for the ERWiN internship.
